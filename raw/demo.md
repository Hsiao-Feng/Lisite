## 欢迎使用
欢迎您使用 Lisite ！

Lisite 是一款简约的站点生成程序，可以从 Markdown 文档生成站点页面。

要使用 Lisite，您需要使 Markdown 的前 5 行是用于 Lisite 解读的信息，遵循以下格式：

```md
<!--
标题
YYYY/MM/DD hh:mm
描述
-->
```

例如：

```md
<!--
欢迎使用 Lisite
2022/05/19 12:00
一篇使用教程
-->
```

此外，您需要使文档以`.md`结尾，以便程序识别。

祝您使用顺利。
